# Mairie Port-de-Paix Desktop 1.2.2

## Correctifs

- **Recensement — responsable urbanisme** : suppression de l’erreur 403 au chargement des projets de construction (appels admin marchés/cimetières remplacés par les routes staff ou supprimés quand inutiles).
- **Navigation recensement** : l’urbaniste ne voit plus que ses rubriques (projets de construction, domaine de l’État, propriétés bâties, écoles, cimetières, églises).
- **Moniteur géographique** : même filtre de rubriques pour le rôle urbanisme.
- **Messages d’erreur** : libellés en français, sans détails techniques (Forbidden, status HTTP, etc.).

## Installation

Windows : exécuter `Mairie Port-de-Paix-Setup-1.2.2.exe` dans le dossier `release/`.
